NamedSemaphore

open class NamedSemaphore : PGSemaphore

This class implements an named (intra-process) semaphore. This type of semaphore is visible to multiple processes. If you need a semaphore that is visible to multiple processes then use a NamedSemaphore.

  • value

    The current value of the semaphore. If this value is less than or equal to zero then any call to acquire(), tryAcquire(), or tryAcquire(until:) will respectively either block, fail, or potentially timeout until this value becomes greater than zero. Currently, only Linux allows for reading the current value of the semaphore.
