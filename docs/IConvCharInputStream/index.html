Opens the character stream for reading. If the stream has already been opened then calling this method does nothing.

close() 

open func close()

Closes the character stream after which no further characters can be read. If the stream had never been opened it will still go directly into a closed state. If the character stream has already been closed then calling this method does nothing. Once a character stream has been closed it can never be reopened.
