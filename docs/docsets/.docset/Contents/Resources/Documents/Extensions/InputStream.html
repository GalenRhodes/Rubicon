  • read(buffer:)

    Read bytes into an instance of EasyByteBuffer. This method assumes that the EasyByteBuffer.count field is being used to store the number of bytes in the buffer. The newly read bytes will be appended to the end of the existing bytes, as denoted by the value in the EasyByteBuffer.count field. If EasyByteBuffer.count is less than zero (0) or greater than EasyByteBuffer.length then a fatalError is thrown. If EasyByteBuffer.count is equal to EasyByteBuffer.length then this method returns immediately with the value zero (0).

    Declaration

    Swift

    public func read(buffer b: EasyByteBuffer) -> Int

    Parameters

    b

    the EasyByteBuffer that will be used to store the bytes read.

    Return Value

    the number of bytes read into the buffer or zero (0) if the buffer is full or the stream it at EOF or -1 if there was an I/O error.
