public func NanoSleep(seconds: UInt32 = 0, nanos: UInt32 = 0) throws

The NanoSleep(seconds:nanos:) function causes the calling thread to sleep for the amount of time specified in the seconds and nanos parameters (the actual time slept may be longer, due to system latencies and possible limitations in the timer resolution of the hardware). An unmasked signal will cause NanoSleep(seconds:nanos:) to terminate the sleep early, regardless of the SA_RESTART value on the interrupting signal.

Parameters

Name Type Description
seconds UInt32

the number of seconds to sleep.
nanos UInt32

the number of additional nanoseconds to sleep.

Throws

CErrors.EINVAL(description:) if nanos was greater than or equal to 1,000,000,000.
