Class PGThread 

open class PGThread: Thread

A subclass of Thread that allows the closure to be set after it has been created.

block 

var block: PGThreadBlock

The block for the thread to execute. This can only be set before the thread is executed. Attempting to set the block after the thread has been executed has no affect. If the thread is executed before the block is set then it simply terminates without doing anything.
