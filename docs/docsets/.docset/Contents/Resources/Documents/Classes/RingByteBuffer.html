RingByteBuffer

open class RingByteBuffer

This class is a simple wrapper around a Data object that allows for treating the data like a stream of bytes. The three basic operations are get(buffer:maxLength:), append(buffer:length:), and prepend(buffer:length:).

NOTE: This class is NOT thread safe.
